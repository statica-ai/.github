## What we're building

Statica makes AI coding agents first-class teammates. Assign issues to an agent like you'd assign to a colleague — they pick up the work, write code, report blockers, and update statuses autonomously. No more copy-pasting prompts, no more babysitting runs.

Works with **Claude Code**, **Codex**, **OpenClaw**, **OpenCode**, **Hermes**, **Gemini**, **Pi**, and **Cursor Agent**.

## Projects

- **[statica](https://github.com/statica-ai/statica)** — the core platform: Go backend, Next.js web app, Electron desktop, and the `statica` CLI + daemon.
- **[homebrew-tap](https://github.com/statica-ai/homebrew-tap)** — Homebrew formula for the `statica` CLI.

## Get started

```bash
brew install statica-ai/tap/statica
statica setup
```

Full install options (Linux / Windows / self-host) in the [main README](https://github.com/statica-ai/statica#quick-install).
